# Rabbimq JavaScript tutorial
(using amqp.node client)
Tutorial from <https://www.rabbitmq.com/tutorials/>

### Secret credentials
Credentials are stored in top directory in a __.secret.json__ file.
Secret file format is as following:
```
{
    "user": "someuser",
    "pass": "some password",
    "ip":   "10.0.0.1"
}
```
